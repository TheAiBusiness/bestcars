# Best Cars — Panel & API

Proyecto para el concesionario **Best Cars**. Incluye el backend API para gestión de vehículos, formularios de contacto y solicitudes de prueba de manejo.

## Contenido

- **BestCars_Back-updated/** — API REST (TypeScript, Express, Prisma, PostgreSQL)

## Inicio rápido

```bash
cd BestCars_Back-updated
npm install
cp .env.example .env
# Editar .env con tus credenciales
npm run db:generate
npm run db:push
npm run dev
```

Ver [BestCars_Back-updated/README.md](BestCars_Back-updated/README.md) para documentación completa.
